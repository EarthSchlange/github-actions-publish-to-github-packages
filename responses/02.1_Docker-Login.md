1. Open a terminal
2. Use the following command to login:
   ```
   docker login docker.pkg.github.com -u USERNAME -p TOKEN
   ```
3. Replace `USERNAME` with your GitHub username
4. Replace `TOKEN` with the Personal Access Token you just created
5. Press **Enter**
